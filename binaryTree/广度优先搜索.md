对每一层节点依次访问，自上而下，从左到右依次访问节点，用队列实现

```
    3
   / \
  9  20
    /  \
   15   7
```

广度优先搜索输出为：3,9,20,15,7

```js
function bfs(root) {
    const res = [];
    const queue = [];
    queue.push(root);

    while (queue.length) {
        const node = queue.shift();
        res.push(node.val);
        if (node.left) {
            queue.push(node.left);
        }
        if (node.right) {
            queue.push(node.right);
        }
    }
    return res;
}
```
